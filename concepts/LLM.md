# LLM

> Large Language model

- Text input -> Text output 
- Probabilistyczny, niedeterministyczny model językowy
- Model jest funkcją probabilistyczną, a nie deterministyczną
- Twarde formuły, a miekkie wzorce (intucje)

- Znaczenie słów wynika z [Kontekst](./Kontekst.md)

- Nauczony wzorcow językowych (parametry - Miliardów źródeł)
- Płynna gramatycznie poprawna rozmowa [Chat](./Chat.md)
- Dostęp do wiedzy, kontekstu [Asystant](./Asystant.md)
